# Smart Contact Manager

Smart Contact Manager is a web application that helps users manage their contacts securely.  
It is built using **Spring Boot** and follows the **MVC architecture**, handling both frontend and backend operations.

## Features

- User authentication and authorization using **Google OAuth**
- Secure login system with **Spring Security**
- Password and user data protection
- Add, update, delete, and manage contacts
- Store user and contact data in a **SQL database**
- User-friendly interface for managing contacts

## Technologies Used

- Java
- Spring Boot
- Spring Security
- OAuth 2.0 (Google Authentication)
- Thymeleaf
- HTML, CSS, JavaScript
- SQL Database
- MVC Architecture

## Security

This project uses **Spring Security** to secure user authentication and protect sensitive data.  
Passwords are encrypted before storing them in the database.

## Future Improvements

In future updates, the project will include an **AI-powered email assistant** that can automatically generate replies to emails from clients or other contacts.

## Project Purpose

The goal of this project is to demonstrate the implementation of:
- Secure authentication
- OAuth integration
- MVC architecture
- Database management in a real-world application
